AnLex Guard

A professional-grade Raspberry Pi based home security system with motion detection, RFID access control, environmental monitoring, and real-time alerts.

Project Overview

AnLex Guard is a comprehensive home security solution built using Raspberry Pi and various sensors. The system provides multi-mode operation (DISARMED, ARMED, PRE-ALARM, ALARM states) with intelligent motion detection, RFID-based access control, and environmental monitoring capabilities.

Designed with both security and user experience in mind, AnLex Guard features a modern web dashboard for real-time monitoring and control, cloud integration with Adafruit IO for data logging, and instant email notifications through Brevo when security events occur.
